لماذا هذه المرحلة قبل غيرها؟

المرحلة الأولى هي اللبنة الحجرية للمشروع كاملاً. كل مرحلة لاحقة (المالية، الحجب، الطلبات، الخطابات، الأكاديمي اليومي، HR، التواصل الذكي) تستند إلى الأربعة أسابيع هذه. أيّ تنازل هنا يُكلّف 5—10 أضعاف لاحقاً. لهذا تُنفَّذ المرحلة باعتماد Codex 100/100 الصارم على كل قطعة قبل البدء وبعد الانتهاء.

المرحلة تجمع ثلاث طبقات متراكبة: (1) الأساس التقني والمعماري — Laravel 11 + PHP 8.3 + PostgreSQL 15 + Filament 3 + Livewire 3 + TailwindCSS RTL + Spatie Permission + Filament Shield + Audit عبر Spatie Activity Log + CI/CD. (2) منصة الاختبارات الكاملة — بنك أسئلة بـ14 نوع + محرك توليد + كود الاختبار (Remote/Onsite) + التصحيح الآلي (Laravel Services + Jobs) + الشهادات بـQR (Bacon QR + Browsershot) + PWA. (3) دورة حياة الطالب — Model موسّع + State Machine بـ8 حالات (Spatie ModelStates) + نقل بين الفروع + بطاقة الطالب الموحّدة بـ7 tabs + دور مدير الفرع + موظف التسجيل.

الأهداف الكبرى

  • O1. إنتاج نواة معمارية مستقرة تخدم 1,000 طالب موزّعين على 4 فروع، مع جاهزية للتوسّع إلى 10,000 طالب و8 فروع بدون إعادة كتابة.
  • O2. فرض عزل الفروع تقنياً عبر Filament Tenancy + Global Scopes + Spatie Permission — مدير الفرع الأول لن يستطيع تقنياً رؤية بيانات الفرع الثاني.
  • O3. بناء بنك أسئلة كأصل معرفي مستقل يبقى عابراً للمعلمين والدورات والسنوات، مع 14 نوع سؤال + 3 قنوات إدخال.
  • O4. منصة اختبارات تتحمّل 100 طالب متزامن مع تصحيح آلي بدقة 100% (عبر Laravel Queue + Horizon) وشهادات قابلة للتحقق العام بـQR.
  • O5. State Machine حقيقي للطالب (Spatie ModelStates) — كل انتقال له Transition + guard + action + سجل تاريخي كامل.
  • O6. Audit Log غير قابل للحذف لكل عملية حساسة (Spatie Activity Log) + امتثال PDPL مبنيٌّ في الـSchema لا مُضافٌ لاحقاً.

القرارات المعمارية المُعتمدة

  • Multi-Branch by Design: كل جدول يحوي branch_id + BranchScope Global Scope + Filament Tenancy.
  • Laravel 11 + PHP 8.3 — Strict Types في كل ملف PHP + Readonly properties + Enums حقيقية بدل constants.
  • Filament 3 Panel Builder — لوحة موحّدة /admin + Multi-Tenancy لكل فرع.
  • PostgreSQL 15+ على VPS سعودي (Hostinger KSA / Contabo KSA) + Backup يومي.
  • Conventional Commits + Husky pre-commit + GrumPHP + Pint (Laravel Code Style).
  • PWA فقط — لا تطبيق جوال Native، لا متاجر تطبيقات.

declare(strict_types=1);

namespace App\Models;

use Illuminate\Database\Eloquent\Concerns\HasUuids;
use Illuminate\Database\Eloquent\Factories\HasFactory;
use Illuminate\Database\Eloquent\Model;
use Illuminate\Database\Eloquent\Relations\BelongsTo;
use Illuminate\Database\Eloquent\Relations\BelongsToMany;
use Illuminate\Database\Eloquent\Relations\HasMany;
use Illuminate\Database\Eloquent\SoftDeletes;
use Spatie\Activitylog\LogOptions;
use Spatie\Activitylog\Traits\LogsActivity;

class Branch extends Model
{
    use HasFactory, HasUuids, SoftDeletes, LogsActivity;

    protected $fillable = [
        'tenant_id', 'code', 'name_ar', 'name_en',
        'address_full', 'phone', 'manager_user_id',
        'working_hours', 'geo_lat', 'geo_lng',
        'timezone', 'settings', 'is_active',
    ];

    protected $casts = [
        'working_hours' => 'array',
        'settings'      => 'array',
        'is_active'     => 'boolean',
        'geo_lat'       => 'decimal:6',
        'geo_lng'       => 'decimal:6',
    ];

    public function getActivitylogOptions(): LogOptions
    {
        return LogOptions::defaults()
            ->logOnly(['name_ar', 'name_en', 'phone', 'is_active', 'manager_user_id'])
            ->logOnlyDirty()
            ->dontSubmitEmptyLogs();
    }

    public function tenant(): BelongsTo
    {
        return $this->belongsTo(Tenant::class);
    }

    public function manager(): BelongsTo
    {
        return $this->belongsTo(User::class, 'manager_user_id');
    }

    public function users(): HasMany
    {
        return $this->hasMany(User::class);
    }

    public function staff(): BelongsToMany
    {
        return $this->belongsToMany(User::class, 'user_branches')
            ->withPivot('is_primary', 'granted_at', 'granted_by');
    }

    public function students(): HasMany
    {
        return $this->hasMany(Student::class);
    }
}

namespace App\Models\Scopes;

use Filament\Facades\Filament;
use Illuminate\Database\Eloquent\Builder;
use Illuminate\Database\Eloquent\Model;
use Illuminate\Database\Eloquent\Scope;
use Illuminate\Support\Facades\Auth;

class BranchScope implements Scope
{
    public function apply(Builder $builder, Model $model): void
    {
        $user = Auth::user();
        if (! $user) {
            return;
        }

        // super_admin يرى كل شيء
        if (method_exists($user, 'hasRole') && $user->hasRole('super_admin')) {
            return;
        }

        // فرع نشط من Filament Tenancy
        $tenant = Filament::getTenant();
        if ($tenant) {
            $builder->where($model->getTable().'.branch_id', $tenant->getKey());
            return;
        }

        // fallback: قائمة فروع المستخدم
        $branchIds = $user->branches->pluck('id')->all();
        $builder->whereIn($model->getTable().'.branch_id', $branchIds);
    }
}

     * @param  array{topic:string, difficulty:string, type:string, count:int}  $params
     * @return array{questions: array<int, array<string, mixed>>, tokens: int}
     */
    public function suggestQuestions(array $params): array
    {
        $prompt = view('prompts.suggest-questions', $params)->render();

        $response = $this->client->messages()->create([
            'model'      => config('services.anthropic.model'),
            'max_tokens' => 4096,
            'messages'   => [['role' => 'user', 'content' => $prompt]],
        ]);

        $raw = $response->content[0]->text ?? '';
        $parsed = json_decode($raw, true, 512, JSON_THROW_ON_ERROR);

        return [
            'questions' => $parsed['questions'] ?? [],
            'tokens'    => $response->usage->inputTokens + $response->usage->outputTokens,
        ];
    }

            QuestionType::ESSAY->value          => $manual,
            QuestionType::SHORT_ANSWER->value   => $manual,
        ];
    }

    /**
     * @return array{score: float, max: float, details: array<string, mixed>}
     */
    public function gradeQuestion(Question $q, mixed $response, float $maxMarks): array
    {
        $grader = $this->graders[$q->type->value] ?? null;
        if (! $grader) {
            return ['score' => 0, 'max' => $maxMarks, 'details' => ['error' => 'unknown_type']];
        }

        return $grader->grade($q, $response, $maxMarks);
    }

namespace App\Services\Exams\Graders;

use App\Models\Question;

class McqMultiGrader implements GraderInterface
{
    public function grade(Question $q, mixed $response, float $maxMarks): array
    {
        $correct  = collect($q->answer_data['correct_option_ids'] ?? []);
        $selected = collect($response['selected_ids'] ?? []);
        $method   = $q->answer_data['scoring_method'] ?? 'all_or_nothing';

        $correctSelected = $selected->intersect($correct)->count();
        $wrongSelected   = $selected->diff($correct)->count();
        $totalCorrect    = $correct->count();

        $score = match ($method) {
            'all_or_nothing' => ($correctSelected === $totalCorrect && $wrongSelected === 0)
                ? $maxMarks : 0,
            'partial' => max(0.0, ($correctSelected / max(1, $totalCorrect)) * $maxMarks),
            'penalty' => max(0.0, (($correctSelected - $wrongSelected) / max(1, $totalCorrect)) * $maxMarks),
            default   => 0,
        };

        return [
            'score'   => round($score, 2),
            'max'     => $maxMarks,
            'details' => compact('correctSelected', 'wrongSelected'),
        ];
    }
}

                    'valid_until' => $until,
                    'is_revealed' => $delivery === 'remote',
                    'max_uses'    => $maxUses,
                ]);
            }

            throw new \RuntimeException('CODE_GENERATION_FAILED');
        });
    }

    private function generateCode(int $length = 6): string
    {
        $alphabet = self::ALPHABET;
        $code = '';
        for ($i = 0; $i < $length; $i++) {
            $code .= $alphabet[random_int(0, strlen($alphabet) - 1)];
        }
        return $code;

            $writer = new Writer($renderer);
            $qrSvg = $writer->writeString($verifyUrl);

                'cert'    => $cert,
                'student' => $student,
                'qr_svg'  => $qrSvg,
            ])->render();

                ->format('A4')
                ->landscape()
                ->showBackground()
                ->waitUntilNetworkIdle()
                ->save(Storage::disk('public')->path($pdfPath));

            // 4) تحديث المسار

            return $cert;

<?php

declare(strict_types=1);

namespace App\Actions\Students;

use App\Models\Student;
use App\Models\StudentTransfer;
use App\States\Student\{FinancialSuspended, DeprivedFees};
use Illuminate\Support\Facades\DB;
use Illuminate\Validation\ValidationException;

class RequestTransferAction
{
    /**
     * @param  array{to_branch_id:string, reason:string}  $input
     */
    public function execute(Student $student, array $input): StudentTransfer
    {
        if ($student->branch_id === $input['to_branch_id']) {
            throw ValidationException::withMessages([
                'to_branch_id' => 'لا يمكن النقل إلى نفس الفرع.',
            ]);
        }

        if (in_array($student->state::class, [FinancialSuspended::class, DeprivedFees::class], true)) {
            throw ValidationException::withMessages([
                'state' => 'لا يمكن نقل طالب موقوف مالياً. يجب سداد المتأخرات أولاً.',
            ]);
        }

        $user = auth()->user();
        $isSuperAdmin = $user->hasRole('super_admin');

        return DB::transaction(function () use ($student, $input, $user, $isSuperAdmin) {
            $transfer = StudentTransfer::create([
                'student_id'     => $student->id,
                'from_branch_id' => $student->branch_id,
                'to_branch_id'   => $input['to_branch_id'],
                'reason'         => $input['reason'],
                'state'          => $isSuperAdmin ? 'approved' : 'pending',
                'requested_by'   => $user->id,
                'approved_by'    => $isSuperAdmin ? $user->id : null,
                'approved_at'    => $isSuperAdmin ? now() : null,
            ]);

            if ($isSuperAdmin) {
                $student->update(['branch_id' => $input['to_branch_id']]);
                $transfer->update(['state' => 'completed', 'completed_at' => now()]);
            }

            return $transfer;
        });
    }
}
